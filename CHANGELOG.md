# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.1] - 2026-02-08

### Fixed

- CI backend test step now skips gracefully when no test files are present.

### Changed

- Added open source launch baseline files: contributing, code of conduct, security, support, issue templates, PR template, and label sync workflow.
- Improved setup/onboarding docs and environment template handling for safer public usage.

## [0.1.0] - 2026-02-07

### Added

- Core pantry domains: stock, recipes, suggestions, and query flows.
- FEFO consumption logic with preview-before-commit cooking flow.
- Barcode scanning integration through OpenFoodFacts.
- Recipe management and stock transaction traceability.

### Changed

- Repository rebranding from previous project naming to PantryOps.
- Branch baseline aligned on `main` and `develop` for the new remote.
