# Support

## Where to Ask for Help

- Usage questions and setup help: open a GitHub Discussion
- Bug reports: open a GitHub Issue with steps to reproduce
- Security concerns: follow `SECURITY.md`

## Before Opening an Issue

- Check existing issues and discussions
- Use the latest code from `main`
- Include logs, environment details, and exact reproduction steps

## Current Scope

PantryOps is an early-stage open source project.

- Focus: core inventory-aware cooking flow
- Not in scope: guaranteed SLA, managed hosting, enterprise support
